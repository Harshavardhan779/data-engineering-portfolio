# Weather Data Analyzer

Simple data analysis project analyzing weather patterns across Indian cities.

## What It Does
- Analyzes temperature and humidity patterns
- Finds average temperature per city
- Identifies most common weather conditions
- Locates hottest day and most humid city

## Technologies
- Python
- Pandas
- CSV data processing

## Files
- `weather_data.csv` - Input data (5 days, 3 cities)
- `weather_analyzer.py` - Analysis script
- `weather_summary.csv` - Output summary

## What I Learned
- Working with weather data
- Data aggregation by city
- Finding max/min values
- Creating summaries
